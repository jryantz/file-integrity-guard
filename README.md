## Installation

### Windows

Create a virtual environment:

```
python -m venv env
```

Activate the environment:

```
env/Scripts/activate
```

Install requirements:

```
pip install -r requirements.txt
```

### Mac

Create a virtual environment:

```
python3 -m venv env
```

Activate the environment:

```
source env/bin/activate
```

Install requirements:

```
pip install -r requirements.txt
```
